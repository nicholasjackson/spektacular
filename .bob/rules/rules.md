# Bob Rules

## Mandatory Documentation Query

Before starting ANY task that involves:
- Writing or modifying code
- Creating policies (OPA, security, etc.)
- Implementing features
- Debugging or fixing issues
- Working with specific frameworks or libraries

You MUST:
1. First use the `query_library_docs` tool with relevant search terms
2. Review the documentation results
3. Only then proceed with the task implementation

Do NOT ask for permission - just use the tool immediately.
Do NOT explain you're using it - just use it.

If the tool returns no results or fails, proceed with the task normally.
